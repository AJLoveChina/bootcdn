## 前言
如果可以，请选择jsdelivr，更好更强更专业的免费CDN！ https://github.com/AJLoveChina/bootcdn/issues/1

## 安装

```bash
$ npm install -g bootcdn
```

## 如何使用

1.查询不带版本
```bash
$ bootcdn jquery
```

2.查询带版本
```bash
$ bootcdn jquery 1.10
```

3.查询所有的版本
```bash
$ bootcdn jquery all
```

3.帮助
```bash
$ bootcdn -info
```
