# bootcdn

快速查询一个开源库在bootcdn的URL地址

<a href="https://asciinema.org/a/UFYyfYoULErlqDjVq7ggxDZW3" target="_blank"><img src="https://asciinema.org/a/UFYyfYoULErlqDjVq7ggxDZW3.svg" /></a>

## 安装

```bash
$ npm install -g bootcdn
```

## 如何使用

1.查询不带版本
```bash
$ bootcdn jquery
```

2.查询带版本
```bash
$ bootcdn jquery 1.10
```

3.查询所有的版本
```bash
$ bootcdn jquery all
```

3.帮助
```bash
$ bootcdn -info
```
